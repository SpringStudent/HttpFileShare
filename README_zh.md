#### 意图

为了便捷的将文件分享出去，该项目由此诞生了

* 一键将本地文件夹通过http协议分享！
* 无需通过ftp或者微信服务器中转文件。

#### 参数说明

| 参数                         | 描述                                |
|----------------------------|-----------------------------------|
| -DhttpFileShare.port=11111 | 指定http服务端口号，如未指定则系统随机获取一个未被占用的端口号 |
